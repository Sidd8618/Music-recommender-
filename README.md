# Music Recommender System

This project demonstrates the basics of Machine Learning by building a music recommendation system that predicts a user's preferred music genre based on age and gender. It was developed using Python, Jupyter Notebook, Pandas, and Scikit-Learn.

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Scikit-Learn
* Joblib

## Project Pipeline

1. Load and explore the dataset.
2. Prepare input features (age, gender) and target labels (music genre).
3. Split the data into training and testing sets.
4. Train a Decision Tree Classifier.
5. Make predictions for new users.
6. Evaluate model accuracy.
7. Save the trained model for future use.
8. Visualize the decision tree to understand the model's decisions.

## Key Learning Outcomes

* Data preprocessing and analysis
* Supervised Machine Learning
* Model training and evaluation
* Model persistence using Joblib
* Decision Tree visualization
